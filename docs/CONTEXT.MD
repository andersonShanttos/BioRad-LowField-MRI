# CONTEXTO — Projeto BioRad (resumo rápido para novos chats)

Fases do Projeto:
• Fase 1 (Mestrado – mai/out 2026): Halbach + B0 + homogeneidade + documentação (sem RF; bobina passiva opcional).
• Fase 2 (Extensão – nov/26–abr/27): Bobina TX/RX completa + matching + RF básico (PA 30–60 W).
• Fase 3 (Extensão/P&D – mai/out 2027): Red Pitaya + Pulseq + MaRCoS → aquisição (FID/SE), ISMRMRD → BART/SIRF, QA (phantoms).
• Fase 4 (Extensão – nov/27–abr/28): Energia portátil, itinerância, logística, POPs de campo.

Cronograma geral:
• Extensão começa mar/2026.
• Mestrado começa mai/2026 (Fase 1).
• Replicação futura: 3–4 protótipos (pós Fase 4).
